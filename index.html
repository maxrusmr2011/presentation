Material-UI

The world's most popular React UI framework.

React components for faster and easier web development.

Installation
As an npm package 

              npm install @material-ui/core
or 

              yarn add @material-ui/core
Material-UI use Roboto fonts
In order to use the font Icon components
In order to use prebuilt SVG Material icons 

              npm install @material-ui/icons
or 

              yarn add @material-ui/icons
using Material-UI with minimal Front-end infrastructure 

              https://unpkg.com/@material-ui/core@latest/umd/material-ui.development.js
or 

              https://unpkg.com/@material-ui/core@latest/umd/material-ui.production.min.js

Features

  • Material-UI components work in isolation (They don't rely on any global style-sheets)
  • using some CSS styles as attributes

Box

wrapper for component

with DIV tag 

            <Box>
              components...
            </Box>
with other tag (for example - span) 

            <Box component="span">
              components...
            </Box>

Container

centers you content

It is bounded by max-width 

            <Container maxWidth="sm">
              <MyComponent />
            </Container>
It has fixed width 

            <Container fixed>
              <MyComponent />
            </Container>

Grid

adaptive layout based on flexbox grid


            <Grid>
              <MyComponent1 />
              ...
            </Grid>
Attributes:
  • container, item - for flex-container or flex-item
  • spacing = { N } - spacing width (0 - 10)
  • [xs/sm/md/lg/xl] = { N } - item size ( 1 - 12 )
  • direction = "value" - css value: flex-direction
  • justify = "value" - css value: justify-content
  • alignItems = "value" - css value: align-items

  • only = 'xs', only = {['xs', 'lg']} - only breakpoints
